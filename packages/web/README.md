# Comply AI — Web Dashboard

## Clean Start (No Demo Data)

Add to `.env.local`:

```
NEXT_PUBLIC_DEMO_MODE=false
```

Restart the dev server. All pages will start with empty state.

By default, `DEMO_MODE` is `true` — the dashboard is pre-populated with sample AI systems, documents, evidence, risk assessments, conformity checklists, monitoring data, incidents, and discovery scans for evaluation purposes.
